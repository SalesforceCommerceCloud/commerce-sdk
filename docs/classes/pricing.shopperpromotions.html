Class ShopperPromotions

Shopper Promotions

View details for active promotions.

Simple example:

  import { Pricing, ClientConfig } from "commerce-sdk";
  // or
  const { Pricing, ClientConfig } = require("commerce-sdk");

  const clientConfig: ClientConfig = { ... };
  const shopperPromotionsClient = new Pricing.ShopperPromotions(clientConfig);

Example with shopper auth

const clientConfig: ClientConfig = {
  parameters: {
    clientId: "XXXXXX",
    organizationId: "XXXX",
    shortCode: "XXX",
    siteId: "XX"
  }
};
token = await helpers.getShopperToken(clientConfig, { type: "guest" });
clientConfig.headers["authorization"] = token.getBearerHeader();
const shopperPromotionsClient = new Pricing.ShopperPromotions(clientConfig);

In the request URL, you must provide a campaign_id parameter, and you can optionally specify a date range by providing start_date and end_date parameters. Both parameters are required to specify a date range, as omitting one causes the server to return a MissingParameterException fault.
